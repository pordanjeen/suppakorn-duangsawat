About CSS

CSS (Cascading Style Sheets) is used to design and control the layout, colors, fonts, and spacing of a website.

CSS makes websites attractive and user-friendly.

What You Can Do with CSS

  • Change text color and size
  • Add backgrounds and borders
  • Create layouts and animations
  • Make responsive designs

Example

Simple CSS code:

p { color: green; font-size: 18px; }
